
# 🩺 Chronic Kidney Disease (CKD) Analysis & Prediction
## 📌 Project Overview

This project focuses on analyzing Chronic Kidney Disease (CKD) using  exploratory data analysis. The goal is to perform data preprocessing, exploratory data analysis (EDA), and build predictive models to classify whether a patient has CKD or not.
This notebook demonstrates a complete end-to-end data science pipeline including:
- Data cleaning
- Handling missing values
- Feature engineering
- Univariate Analysis
- Bivariate Analysis
- Multivariate analysis

## 📊 Project Workflow
### 1️⃣ Data Preprocessing
- Handled missing values

- Cleaned inconsistent data entries

### 2️⃣ Exploratory Data Analysis (EDA)
- Distribution analysis
